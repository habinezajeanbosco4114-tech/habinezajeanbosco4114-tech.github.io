C++ Programming Notes

1. Introduction to C++

C++ is a programming language used to build software, games and systems.

2. Basic Structure of C++ Program

#include 
using namespace std;

int main() {
    cout << "Hello World";
    return 0;
}

3. Variables

Common Data Types:

  • int – Integer numbers
  • float – Decimal numbers
  • double – Large decimal numbers
  • char – Single character
  • string – Text

Example:

int age = 20;
float salary = 500.5;
string name = "John";

4. If Statement

int number = 10;

if(number > 5) {
    cout << "Number is greater than 5";
}

5. Loops

For Loop

for(int i = 1; i <= 5; i++) {
    cout << i << endl;
}

While Loop

int i = 1;
while(i <= 5) {
    cout << i;
    i++;
}
