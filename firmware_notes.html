1. What is Firmware?

Firmware is low-level software embedded in hardware devices.

2. Firmware Example (Embedded C)

#include 
int main(void) {
    DDRB |= (1 << PB0);
    while(1) {
        PORTB ^= (1 << PB0);
    }
}

3. Firmware Updating Steps

  • Download firmware file
  • Connect device
  • Flash firmware
  • Restart device
