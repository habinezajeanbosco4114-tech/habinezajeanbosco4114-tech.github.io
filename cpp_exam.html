Advanced C++ Examination

Section A: Theory

1. What is OOP?

Answer: Object-Oriented Programming is a programming paradigm based on objects and classes.

2. Define Encapsulation.

Answer: Encapsulation is hiding data using private members.

3. What is Inheritance?

Answer: Inheritance allows one class to inherit properties from another.

5. Write a program demonstrating inheritance.

#include 
using namespace std;

class Parent {
public:
    void show() {
        cout << "Parent class";
    }
};

class Child : public Parent {
};

int main() {
    Child c;
    c.show();
}
